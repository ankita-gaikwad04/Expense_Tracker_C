# Expense Tracker in C

A simple Expense Tracker project developed using C programming language.  
This project stores expense records in a CSV file and helps users manage daily expenses.

## Features
- Add expenses
- Store data in CSV file
- View expense records
- Simple menu-driven program

## Technologies Used
- C Programming
- File Handling
- CSV Storage

## How to Run
1. Compile the code
2. Run the executable
3. Enter expense details
